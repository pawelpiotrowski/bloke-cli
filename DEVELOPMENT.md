# Bloke CMS CLI App

Generated by [create-pastel-app](https://github.com/vadimdemedes/create-pastel-app).

## Install Locally

```bash
$ npm install . -g
```

## Available Commands

Commands are mapped from `source/commands` directory.
For details run:

```bash
$ blokecms-cli --help
```

## Use with npx

```bash
$ npx blokecms-cli@latest
# or
$ npm cache clean --force
$ npx clear-npx-cache
$ npx blokecms-cli
```

## Package Scripts

```bash
# build
$ npm run build

# watch and rebuild
$ npm run dev

# run unit tests
$ npm run test

# update app version (major,minor,path or semver string)
$ npm run version {ARG}

#output current version
$ npm run version show
```
